/^>[^\&\s].+/g: green text
Observation: it will be run against individual lines of the message, not against the whole message at once.
Class: greenText
Example: 
>text

/\[spoiler\].+\[\/spoiler\]/g: spoiler
Observation: it will be run against individual lines of the message, not against the whole message at once.
Class: spoiler
Example:
[spoiler]text[/spoiler]

/>>>\/\w+\/\d+/g: cross-quote
Class: quoteLink
Example:
>>>/board/1

/>>>\/\w+\//g: board link
Example:
>>>/board/

/>>\d+/g: quote
Class: quoteLink
Example:
>>1

/(http|https)\:\/\/\S+/g: links
Example:
http://url.com
